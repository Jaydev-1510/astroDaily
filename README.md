# Astro Daily

An archive + gallery of NASA's astronomy picture of the day.

> Get images from space everyday!

# About

Please check out https://apod.NASA.gov

# Getting Started

- Clone the repo:

```bash
git clone https://github.com/Hack-Club-APS/hackclubaps.git
cd hackclubaps
```

- Install dependencies:

```bash
bun install
```

- Run locally:

```bash
bun run dev
```

- Open `http://localhost:4321` in your browser.
